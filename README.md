# VegeFoods
Online Website for Ordering food. Both for the cutomer and the restaurant owner.

An Online Ecommerce Website made using PHP framework.

To run just start the Xampp Server and open the index.html file.
Functionalities:-
1. Cart 
2. Login 
3. Signup
4. Product Display
5. Product Order.
